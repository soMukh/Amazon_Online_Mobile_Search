Case Study : Online Mobile Search

Suggested Site : https://www.amazon.in

Detailed Description :
1.Launch the browser using the configuration settings Firefox/Chrome.
2.Read the application URL from configuration settings. (e.g. https://www.amazon.in)
3.Open the URL. User will navigate to home page of website.
4.Enter the search text in search box “mobile smartphones under 30000”
5.Application displays the follow message similar to following-
o    1-24 of over 1,000 results for "mobile smartphones under 30000"  (1-24 and 1000 numbers will change according to stock available on site that at the time of execution)
6.In the above text, validate the search string, number of pages( e.g. 1-16), number of items (over 90,000)
7.Click on “Sort by list” listbox.
8.Application should display five sort by options in the list. Check the count of options displayed.
9.Select option “Newest Arrivals”
10.Verify that  “Newest Arrivals” option got selected correctly or not.
11.Close the browser.
